# 🔮 Натальная карта (Джьотиш)

Расчёт натальной карты по ведической астрологии. Определяет Лагну, положение планет, накшатру Луны, Вимшоттари-дашу и 18 варг (D1–D60).

## Возможности

- 🏙 **Автоопределение города** — по названию находит координаты и часовой пояс
- 🌙 **Накшатра и даши** — накшатра Луны, управитель, баланс Вимшоттари-даши
- 📊 **18 варг** — от D1 (Раши) до D60 (Шаштьямша)
- 🤖 **Telegram-бот** — пошаговый ввод, результат приходит .md файлом

## Установка

```bash
git clone <repo>
cd natal
uv sync
```

### Настройка бота

Создай файл `.env` в папке `natal/`:

```
BOT_TOKEN=твой_токен_от_BotFather
```

## Запуск

**CLI-версия:**
```bash
uv run main.py
```

**Telegram-бот:**
```bash
uv run bot.py
```

## Использование

### Пошагово (рекомендуется)

1. Напиши боту `/start`
2. Введи город рождения
3. Введи дату (ДД.ММ.ГГГГ)
4. Введи время (ЧЧ:ММ)
5. Получи .md файл с полной картой

### Одной строкой

```
Москва; 15.06.1990; 14:30
```

